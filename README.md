# 🔐 Routage de Données Sensibles — Dijkstra

**Projet Recherche Opérationnelle — ENSAM Casablanca 2025**

## Description
Modélisation d'un réseau informatique sous forme de graphe pondéré.
Application de l'algorithme de Dijkstra pour trouver le chemin optimal
combinant latence et niveau de risque de sécurité.

## Formule de coût
w(u,v) = Latence + 10 × Risque

## Résultat
Chemin optimal : Source → A → D → Dest (coût = 120)

## Technologies
Python · NetworkX · Matplotlib · Cisco Packet Tracer
