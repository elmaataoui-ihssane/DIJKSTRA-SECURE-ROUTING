# DIJKSTRA-SECURE-ROUTING

# Routage de Données Sensibles — Plus Court Chemin Sécurisé

Projet de Recherche Opérationnelle — ENSAM Casablanca 2025

## Description
Modélisation d'un réseau informatique sous forme de graphe pondéré 
et application de l'algorithme de Dijkstra pour déterminer le chemin 
optimal combinant latence et sécurité.

## Technologies
Python · NetworkX · Matplotlib · Théorie des Graphes
